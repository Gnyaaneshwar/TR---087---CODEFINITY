01 / PIPELINE

11-Step Modular Architecture

Each module is independently testable, replaceable, and optimised.

01
INPUT

Input Validation & Normalisation

Accepts file paths or NumPy arrays. Validates identical H×W resolution. Fills missing depth via Euclidean distance transform (scipy). Scales RGB to [0,1] and depth via min-max normalisation.

scipy.ndimageopencvnumpy
02

Point Cloud Generation

Vectorised pinhole back-projection X=(u−cx)Z/fx, Y=(v−cy)Z/fy for every pixel. Produces coloured (N,3) float32 cloud.

numpyvectorised
03

Scene Segmentation

Depth-threshold floor/wall removal then DBSCAN clustering isolates foreground object clusters.

DBSCANscikit-learn
04
CORE

Grasp Candidate Generation

PCA on k-NN neighbourhood estimates local surface normal per anchor. Four orthogonal roll orientations sampled per anchor. GraspNet-compatible output schema.

PCAKD-Treesurface normals
05

Initial Scoring

Point density, surface normal consistency, centroid proximity — three normalised heuristics combined into confidence [0,1].

heuristicnormalised
06

Collision Checking

Oriented bounding-box (OBB) model for left/right finger bodies and palm zone. KD-tree pre-filter → exact OBB test.

OBBKD-Treescipy
07

Stability Estimation

Local plane fitting via SVD. Scores approach-vector alignment with surface normal and contact symmetry.

SVDgeometry
08
RANKING

Hybrid Re-Ranking

Weighted fusion: final_score = 0.4×confidence + 0.3×stability + 0.3×collision_free. Top-K selected and assigned final ranks.

0.4 × Confidence + 0.3 × Stability + 0.3 × Collision-Free
weighted scoringtop-K
09

Physics Validation

Headless PyBullet: lift test + inter-body collision check per grasp. Graceful no-op when not installed.

PyBulletsimulation
10

JSON Output

Structured export: position, orientation (r/p/y), all scores, collision status, physics result, timing metadata.

JSONstructured
11
VIZ

3D Visualization

Open3D (with Matplotlib fallback): point cloud + gripper overlays. Green = best, yellow = valid, red = collision.

Open3DMatplotlib
